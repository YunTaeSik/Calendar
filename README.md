## Calendar
  
  
**프로젝트 기간**  
2016.10 ~ 2016.11

**소속**  
한성대학교 프로젝트
  
**담당역할**  
Calendar Android App 개발 (70% 담당) 

**주사용 기술**  
RecyclerView, Sqlite 적용, UI 적용, 간단한 Animation 적용 등등
  
  
**프로젝트 내용**  
한성대학교 Android 수업 프로젝트로 Calendar App을 제작하였습니다.
전체적인 Sqlite 구조 및 RecyclerView의 구조, Animation을 담당하여 개발했습니다.
